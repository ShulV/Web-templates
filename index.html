Улучшение кода

  • Sort HTML attributes:

    1. Установить расширение
    2. Перейти в настрйоки VS Code
    3. Найти настройки расширений и сам AttrsSorter
    4. Найти там settings.json
    5. Вставить туда строчку из описания расширения:

      "attrsSorter.order": ["class", "id", "href", "src", "data-.+", "aria-.+"]
